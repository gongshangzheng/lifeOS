# 项目、竞赛与开源策略


> 主计划 → [三年职业规划](../three-year-career-2026-2028.md)

## 现有项目评估

4段经历（零一智能 + Bouygues + LamCube + 科研压缩）覆盖生成式 AI 全链条，数量质量超过大部分候选人，不需要额外做新项目。

## 竞赛权重

- Kaggle 金/银牌：中等加分，有时间可以打
- 天池 Top 10：小加分，性价比低
- 数学建模：算法岗基本不看
- 没有竞赛不会成为短板

## 开源建议

把科研代码整理成 GitHub repo > 打竞赛。实习结束后花 1-2 周清理代码，再在 ComfyUI 生态提 1-2 个 PR。

## 时间分配

| 事项 | 占比 |
|------|------|
| 钉钉实习 | 50% |
| LeetCode | 20% |
| 项目梳理 | 10% |
| 科研 | 10% |
| 开源 | 5% |
| 竞赛 | 5% |
