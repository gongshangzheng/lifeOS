### 基本原则（ACSM 立场声明）

1. **渐进超负荷**：连续两次训练能超过目标次数 1-2 次 → 负荷增加 2-10%
2. **大肌群先于小肌群，多关节先于单关节**
3. **增肌区间**：6-12 RM，组间休息 1-2 分钟，中等速度（向心 1-2s，离心 1-2s）
4. **训练频率**：新手每周 2-3 次/部位（每次全身或上下肢分化），中阶每周 3-4 次
5. **周期化**：系统调整训练量与强度是长期进步最有效的方法

> 参考：ACSM Position Stand — Progression Models in Resistance Training for Healthy Adults (2009)

## 二、减脂增肌并行的原理

### 2.1 干扰效应（Interference Effect）

> "有氧不能做太狠，不然肌肉长不了"——这句话背后的科学。

当力量训练和耐力训练在同一周期内并行时，耐力训练会抑制力量和肌肉增长的适应性。这一现象称为**干扰效应**（Hickson, 1980）。

**分子机制**：

- 力量训练 → 激活 **mTOR 通路**（肌肉蛋白合成的主开关）
- 耐力训练 → 激活 **AMPK 通路**（细胞能量传感器）
- AMPK 激活后会**抑制 mTOR**，从而减弱肌肉合成信号

简单说：mTOR 管"建肌肉"，AMPK 管"烧能量"。两者信号打架时，耐力优先。

**Meta-analysis 数据**（Wilson et al., 2012 — pooling 21 studies）：

| 指标 | 合并训练 vs 纯力量 | 下降幅度 |
|------|---------------------|----------|
| 力量增长 | ↓ | ~31% |
| 肥大增长 | ↓ | ~18% |

**但干扰有条件**：

| 因素 | 低干扰 | 高干扰 |
|------|--------|--------|
| 模式 | 骑行 | 跑步（离心冲击大） |
| 频率 | ≤3 次/周 | ≥4 次/周 |
| 时长 | <30min | >30min |
| 强度 | Zone 2（低） | HIIT / 高强度 |
| 时序 | 与力量隔天 | 与力量同日紧邻 |

> **结论**：如果目标是减脂 + 增肌并行，把有氧控制在**骑行/椭圆机、中等强度、20-30 分钟以内**，尽量**与力量训练间隔 6 小时以上或分日**，可以大幅降低干扰。

### 2.2 训练顺序的影响

同一天内既做力量又做有氧时：

- **先力量后有氧**：mTOR 信号激活更充分，对增肌更友好
- **先有氧后力量**：力量表现下降，mTOR 激活减弱

> 参考：Wang et al. (2019), *The order of concurrent training affects mTOR signaling but not mitochondrial biogenesis in mouse skeletal muscle.* PMC7153037

## 三、动作库

> 每个动作列出目标肌群、关键要点和常见错误。
> 推荐的外部动作图谱：[MuscleWiki](https://musclewiki.com)（交互式肌肉图谱，点击肌群看动作视频）、[Muscle & Strength](https://www.muscleandstrength.com/exercises)（1500+ 动作视频库）。

### 3.1 背部

#### 杠铃划船（Barbell Row）

| 项目 | 内容 |
|------|------|
| 目标肌群 | 背阔肌、大圆肌、菱形肌、斜方肌中下束、三角肌后束 |
| 组数 × 次数 | 3 × 8-10 |
| 关键要点 | 髋屈 45°，背挺直；杠铃沿大腿拉至下腹；肩胛先收再拉；离心 2-3s 控制 |
| 常见错误 | 用手臂拉代替背部启动；腰部弯曲；弹借惯性 |

#### 高位下拉（Lat Pulldown）

| 项目 | 内容 |
|------|------|
| 目标肌群 | 背阔肌、肱二头肌、前臂 |
| 组数 × 次数 | 3 × 10-12 |
| 关键要点 | 宽握略宽于肩；先沉肩再下拉；拉至锁骨位置；肘部向下向后 |
| 常见错误 | 下拉时身体过度后仰；用惯性甩；肩胛没收紧 |

#### 坐姿划船（Seated Cable Row）

| 项目 | 内容 |
|------|------|
| 目标肌群 | 背阔肌、菱形肌、斜方肌中束、肱二头肌 |
| 组数 × 次数 | 3 × 12-15 |
| 关键要点 | 脊柱中立位；肩胛后缩带动拉；拉至腹部；肘部贴身体 |
| 常见错误 | 前后晃动借力；过度伸展腰椎；含胸圆肩 |

#### 引体向上（Pull-up）

> 当前无法完成标准引体向上，使用 [渐进训练方案](./pull-up-progression.md)。

### 3.2 手臂

#### 哑铃弯举（Dumbbell Bicep Curl / 二头弯举）

| 项目 | 内容 |
|------|------|
| 目标肌群 | 肱二头肌（长头 + 短头） |
| 组数 × 次数 | 3 × 10-12 |
| 关键要点 | 上臂固定不动；肘部为唯一活动关节；顶端挤压 1s；离心缓慢 2-3s |
| 常见错误 | 借身体晃动甩起；肘部前后移动；下放太快 |

#### 锤式弯举（Hammer Curl）

| 项目 | 内容 |
|------|------|
| 目标肌群 | 肱肌（brachialis）、肱桡肌、前臂 |
| 组数 × 次数 | 3 × 12-15 |
| 关键要点 | 哑铃中立握（像握锤子）；手腕保持中立；上臂固定 |
| 常见错误 | 手腕弯曲；借惯性；大臂前后晃动 |

> **锤式弯举 vs 二头弯举的区别**：锤式弯举更强壮地刺激肱肌和前臂，让手臂看起来更粗；二头弯举更集中刺激肱二头肌的肌峰。两者搭配训练效果最佳。

### 3.3 胸部

#### 平板卧推（Bench Press）

| 项目 | 内容 |
|------|------|
| 目标肌群 | 胸大肌、三角肌前束、肱三头肌 |
| 组数 × 次数 | 4 × 8-12 |
| 关键要点 | 肩胛后缩下沉；杠铃下放至乳头水平；脚踩地稳住；臀部不离开凳面 |
| 常见错误 | 肩膀前伸（前束代偿）；弹胸借力；手腕后翻 |

#### 上斜哑铃卧推（Incline Dumbbell Press）

| 项目 | 内容 |
|------|------|
| 目标肌群 | 胸大肌上束、三角肌前束 |
| 组数 × 次数 | 3 × 10-12 |
| 关键要点 | 凳面 30-45°；哑铃下放至胸口两侧；推至上方但不碰撞 |
| 常见错误 | 角度太大（变成肩推）；底部不做暂停 |

### 3.4 腿部

#### 深蹲（Squat）

| 项目 | 内容 |
|------|------|
| 目标肌群 | 股四头肌、臀大肌、腘绳肌、核心 |
| 组数 × 次数 | 4 × 8-10 |
| 关键要点 | 脚与肩宽；膝盖跟脚尖方向一致；蹲至大腿水平或更低；核心收紧 |
| 常见错误 | 膝盖内扣；脚跟离地；腰部弯曲 |

#### 罗马尼亚硬拉（Romanian Deadlift）

| 项目 | 内容 |
|------|------|
| 目标肌群 | 腘绳肌、臀大肌、竖脊肌 |
| 组数 × 次数 | 3 × 10-12 |
| 关键要点 | 膝盖微曲固定；髋关节后推；背全程挺直；下放至杠铃过膝即可 |
| 常见错误 | 弯腰代替屈髋；膝盖过度前移（变成深蹲） |

### 3.5 肩部

#### 哑铃肩推（Dumbbell Shoulder Press）

| 项目 | 内容 |
|------|------|
| 目标肌群 | 三角肌（前束 + 中束）、肱三头肌 |
| 组数 × 次数 | 4 × 8-10 |
| 关键要点 | 坐姿背部有支撑；哑铃从肩侧推至头顶；不锁死肘关节 |
| 常见错误 | 腰部过度前凸；推起时借腿蹬 |

#### 侧平举（Lateral Raise）

| 项目 | 内容 |
|------|------|
| 目标肌群 | 三角肌中束 |
| 组数 × 次数 | 3 × 12-15 |
| 关键要点 | 微曲肘；从侧面向上举至肩平；想象"倒水"动作 |
| 常见错误 | 重量太大借身体晃；前平举代替侧平举 |

## 四、补剂指南

### 4.1 蛋白粉（Whey Protein）

**作用**：补充蛋白质摄入，方便快捷。每勺约 20-25g 蛋白质。

**何时喝**：训练后 30 分钟内 1 勺（非必须，但方便）。日常蛋白质摄入不足时随时补充。

**每日蛋白质目标**：体重 × 1.6-2.2 g/kg（减脂增肌期取高值）。

> 蛋白粉不是"神药"，它只是食物的便捷替代品。如果日常饮食已达标，额外喝蛋白粉意义不大。

### 4.2 肌酸（Creatine Monohydrate）

**要不要买？基于现有证据的判断**：

| 维度 | 结论 |
|------|------|
| 整体效果 | 大多数 Meta-analysis 显示有效，增益幅度小但有统计显著性 |
| 2025 UNSW 试验 | 54 人 12 周试验发现 5g/天 无显著差异（但样本量偏小） |
| 剂量 | 每日 3-5g，无需 loading phase |
| 形式 | 一水肌酸（Creatine Monohydrate）最便宜、最可靠 |
| 新手增益 | 未训练者改善幅度大于有训练基础者 |
| 性别差异 | 男性有显著效果，部分研究显示女性无显著效果 |
| 安全性 | 25 年研究证实安全，最常见副作用是肠胃不适 |

**分子机制**：肌酸在肌肉中合成磷酸肌酸（PCr），是 ATP 快速再生的燃料库。更多的 PCr = 高强度训练中多做 1-2 个 = 更大的训练刺激。

**我的建议**：

- 如果预算允许（每月 ~50 元），可以买。一水肌酸便宜、安全、有长期研究背书
- 不需要 loading phase，每天 5g 随餐服用即可
- 它不是必需品，优先保证训练、饮食和睡眠
- 4-6 周后观察训练表现（推举重量 / 次数）是否有变化，再决定是否长期服用

> **结论**：可以买，但不是急切需求。先把训练规律和蛋白质摄入做到位，肌酸是锦上添花。

### 4.3 其他常见补剂

| 补剂 | 评级 | 说明 |
|------|------|------|
| 咖啡因 | ✅ 有效 | 训练前 30min 3-6mg/kg，提升力量和专注 |
| β-丙氨酸 | ⚠️ 可能有效 | 适合高强度间歇，效果温和 |
| BCAA | ❌ 不必要 | 蛋白质摄入充足时无额外收益 |
| 氮泵（Pre-workout） | ⚠️ 注意成分 | 主要是咖啡因 + β-丙氨酸 + 糖 |
