# 面试准备清单


> 主计划 → [三年职业规划](../three-year-career-2026-2028.md)

## LeetCode 刷题

- 目标：200-300 题，重点中等难度
- 高频：数组/字符串、链表、二叉树、动态规划、回溯、图论
- 必刷专题：滑动窗口、双指针、BFS/DFS、拓扑排序
- 节奏：实习期每天 2-3 题，秋招前 1 个月每天 4-5 题

## 八股文重点

- Transformer 注意力机制
- 扩散模型原理（前向/逆向/采样）
- Stable Diffusion 架构（VAE + UNet + CLIP）
- LoRA 微调原理
- 过拟合/欠拟合、梯度消失/爆炸、BatchNorm
- 优化器对比（Adam vs SGD vs AdamW）
- 损失函数（交叉熵、Focal Loss、对比损失）

## 面试流程（4-5轮）

- 一面：项目深挖 + 八股文（1-1.5h）
- 二面：项目深挖 + 算法设计 + 场景题
- 三面：交叉面（跨部门，侧重深度）
- HR面：职业规划 + 薪资 + 稳定性

## 项目讲述 STAR 法则

1. Situation：背景（1分钟）
2. Task：问题（1分钟）
3. Action：你做了什么（3-5分钟）
4. Result：量化成果
5. Reflection：困难与解决
