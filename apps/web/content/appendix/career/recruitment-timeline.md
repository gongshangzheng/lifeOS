# 招聘时间线与秋招攻略


> 主计划 → [三年职业规划](../three-year-career-2026-2028.md)

## 核心概念

| 概念 | 说明 |
|------|------|
| 提前批 | 正式秋招前 1-2 个月（7-8月），竞争较小但岗位有限 |
| 内推 | 在职员工推荐，可跳过简历筛选直达面试，提前批前 1 个月联系 |
| 暑期实习 | 研二暑假（2027年）是最关键的实习，目标拿 return offer |
| 日常实习 | 非暑期实习，积累经验但不直接关联转正 |

## 内推实操建议

- 研一暑假实习期间：和同事建立良好关系，为 2027 年内推做准备
- 研二上学期：通过 LinkedIn、牛客网联系目标公司员工
- 提前批前 1 个月：主动联系内推人，提交简历
- 内推 ≠ 保送：只是跳过简历筛选，面试表现才是决定性因素
