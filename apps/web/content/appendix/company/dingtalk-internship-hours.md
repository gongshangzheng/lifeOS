# 钉钉实习工作时间与节奏


> 主计划 → [三年职业规划](../three-year-career-2026-2028.md)
> 调研时间：2026.05

## 工作时间

**钉钉当前要求**（2025.4 无招回归后）：
- **早 9:00 到岗**，固定早会
- **午休 13:15 结束**（阿里其他 BU 一般 14:00）
- 晚间有工作总结会
- 整体工时 10-12h/天

**实习生实际情况**：
- 通常 19:00-20:00 可以下班
- 实习生一般不被要求跟到最晚
- 但如果 mentor 晚上还在，压力会有

## 算法岗日常节奏

| 时间段 | 内容 |
|--------|------|
| 09:00-09:30 | 到岗，早会 |
| 09:30-12:00 | 模型训练、实验调试、数据处理 |
| 12:00-13:15 | 午餐+休息 |
| 13:15-18:00 | 核心编码/实验时间 |
| 18:00-19:00 | 灵活，取决于项目进度 |
| 19:00-20:00 | 实习生通常这个时间段下班 |

**算法岗特点**：训练任务的等待时间决定你的节奏。模型跑着的时候可以学东西、刷题、读论文。

## 通勤与生活

| 项目 | 情况 |
|------|------|
| 园区 | 阿里西溪园区（余杭区文一西路） |
| 食堂 | 十几个食堂，实习饭补 50元/天 |
| 班车 | 覆盖杭州主城区 |
| 交通 | 地铁5号线永福站/善贤站 + 公交/班车 |
| 避峰 | 避开 7:30-9:00 文一西路 |

## 实用建议

- 入职前问清楚 mentor 和 leader 的风格，团队差异很大
- 利用训练等待时间刷题/读论文
- 主动汇报进度，避免被动等待分配
