## 一、钉钉在阿里集团内的位置

```
阿里巴巴集团（CEO 吴泳铭）
├── 阿里中国电商集团
├── 阿里国际数字商业集团
├── 阿里云智能集团
└── Alibaba Token Hub (ATH) 事业群  ← 2026年3月新成立
    ├── 通义实验室（基础模型研发）
    ├── MaaS 业务线（模型即服务）
    ├── 千问事业部（C端AI应用）
    ├── 悟空事业部（B端AI原生工作平台，钉钉升级版）
    └── AI创新事业部
```

**钉钉归属变迁**：
- 2019年：纳入阿里云（"云钉一体"）
- 2023年："1+6+N"改革，拆分独立
- 2025年4月：陈航（无招）回归任CEO，内部重组
- 2025年8月：发布AI钉钉1.0，全面AI化重构
- 2026年3月：并入ATH事业群

## 二、钉钉内部 AI 相关团队

| 团队 | 定位 | 核心产品/能力 |
|------|------|--------------|
| 数据工程团队 | AI训练数据治理 | 数据清洗、向量化、知识库 |
| 模型训练团队 | 垂直领域模型训练 | AI客服大模型、领域微调 |
| 效果评测团队 | AI效果保障 | 准确率评测、A/B测试 |
| AI Native产品团队 | Agent驱动的产品设计 | 钉钉One、AI秘书、AI闪记 |
| AI Fusion | 多模型调度 | 集成50+大模型，智能选模型 |
| 音视频AI | 语音识别/转写 | AI听记（1亿小时音频训练） |

### 音频 Agent 在钉钉的位置

```
钉钉/悟空
├── AI产品线
│   └── AI听记/语音助手 ← 音频Agent核心
├── 音视频事业部
│   ├── 音视频基础能力（RTC、会议）
│   └── 音频AI能力（ASR、TTS、音频Agent）
└── 硬件产品
    └── DingTalk A1（录音硬件，配合音频Agent）
```

音频Agent是钉钉AI战略重点，AI听记项目投入超过10亿元。

## 四、华为整体组织架构

```
华为技术有限公司
├── 轮值董事长制度
├── 三大BG（业务集团）
│   ├── 运营商BG（电信运营商客户）
│   ├── 企业BG（政企客户）
│   └── 消费者BG（终端消费者）
├── Cloud BU（云计算，一级部门）
├── 海思（芯片设计，一级部门）
├── ICT产品与解决方案
│   ├── 无线网络产品线
│   ├── 核心网产品线
│   ├── 光产品线
│   ├── 数据通信产品线
│   ├── 计算产品线 ← 
│   └── 存储产品线
├── 2012实验室（基础研究）
└── 数字能源公司 / 引望公司（独立运营）
```

| BG | 定位 | 核心产品 |
|----|------|----------|
| 运营商BG | 连接管道 | 无线网络、核心网、光传输 |
| 企业BG | 数字化转型 | 数通、存储、智能协作 |
| 消费者BG | 智能终端 | 手机、PC、鸿蒙生态 |
| Cloud BU | 云基础设施 | 华为云、AI云服务 |

## 五、华为计算产品线

```
ICT产品与解决方案
└── 计算产品线
    ├── 鲲鹏产业（通用算力）
    │   ├── 鲲鹏处理器设计
    │   ├── 服务器整机方案
    │   ├── openEuler社区运营
    │   └── 鲲鹏应用迁移适配
    ├── 昇腾产业（AI算力）
    │   ├── 昇腾AI芯片
    │   ├── Atlas硬件（训练卡/推理卡/服务器）
    │   ├── CANN软件栈（算子开发/编译器）
    │   ├── MindSpore AI框架
    │   └── ModelArts（AI开发平台）
    ├── 云与计算硬件部
    └── 生态与解决方案
```

### 五大根技术

| 根技术 | 定位 | 核心产品 |
|--------|------|----------|
| 鲲鹏 | 通用服务器算力 | 鲲鹏920处理器、泰山服务器 |
| 昇腾 | AI算力 | 昇腾910/310芯片、Atlas系列 |
| 欧拉(openEuler) | 开源操作系统 | 服务器OS、边缘OS |
| CANN | AI计算架构 | 算子开发、图编译、异构计算 |
| 昇思(MindSpore) | AI框架 | 全场景深度学习框架 |

### AI 相关岗位

| 岗位 | 所属方向 | 职责 |
|------|---------|------|
| AI工程师 | 昇腾 | 大模型训练、CV |
| 芯片设计工程师 | 昇腾/鲲鹏 | ASIC设计、验证 |
| 嵌入式软件工程师 | CANN/欧拉 | 算子开发、OS内核 |
| 算法工程师 | 昇腾 | AscendC算子开发、性能优化 |

### 实习生分布

- **工作地**：上海、杭州、东莞（松山湖）、西安、成都
- **热门方向**：昇腾（大模型风口）、CANN算子开发、openEuler
- **分配方式**：投递时选ICT-计算产品线，具体小部门由匹配分配
