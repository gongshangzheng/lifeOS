# 音频 Agent 方向与华为实习分析


> 主计划 → [三年职业规划](../three-year-career-2026-2028.md)

## 音频 Agent 方向

音频 Agent（Voice Agent）是以语音为核心交互的 AI 智能体，技术栈覆盖：
- 感知层：ASR（语音识别）、声纹识别、情感计算
- 认知层：LLM 实时推理、意图理解、对话状态管理
- 生成层：TTS（语音合成）、语音克隆、情感化语音生成
- 执行层：Function Calling、工具调用

**市场**：2024年全球 AI 语音市场规模暴涨 187%，2025年企业级 Agent 市场 232 亿元，复合增长率 120%。

**岗位判断**：有前景但需要积累。不像视觉那么卷，也不像纯 LLM 那么热。

**和你背景的关联度约 50%**：
- 扩散模型/自回归模型是通用能力，TTS 也用扩散模型
- Agent 架构（多模块协调、实时推理）是多模态方向的核心能力
- 但音频信号处理、ASR/TTS 是新领域

**实习方向和求职方向不一致在业界很常见**，关键在于叙事能力。

## 华为实习：不需要去

华为实习口碑两极分化（取决于组和 mentor），计算产品线偏向底层算力/芯片使能。

**不需要去的理由**：
1. 已有阿里钉钉实习，简历背书比华为更强
2. 华为计算产品线和生成式 AI/视觉方向差距更大
3. 已有 3 段实习经历，数量够
4. 时间应花在钉钉出成果 + 秋招准备上
