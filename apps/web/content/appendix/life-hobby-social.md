# 生活、爱好与社交


> 主计划 → [三年职业规划](../three-year-career-2026-2028.md)

## 爱好启动

- **日语**：Duolingo → Anki → Comprehensible Japanese YouTube。最低 45min/周
- **绘画**：Drawabox Lesson 0 → Proko YouTube。最低 30min/周
- **徒步**：「两步路」APP，杭州宝石山线起步。每两周一次

## 实习生活节奏

- 工作日：早起30min爱好 → 通勤听播客 → 全力工作 → 步行下班 → 晚上轻松社交
- 周末：半天出门 + 半天宅 + 一天自由
- 找一个「第三空间」打破工位-宿舍循环

## 社交与亲密关系

**内向者社交三法**：成为常客、用可回答的问题开场、做而不是说。

**认识女生的场景**：法语角（你的C1是核武器）、食堂拼桌、豆瓣同城徒步、课题组交叉合作。

**聊天三阶段**：信息交换 → 观点共鸣 → 具体活动邀请（不是邀约，是邀请）。

**心态**：被拒绝是信息不是失败。目标是三个月后多认识三个人、多尝试一个爱好。
