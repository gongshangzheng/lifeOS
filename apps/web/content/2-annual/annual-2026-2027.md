# 2026-2027 详细计划


> 主计划 → [三年职业规划](three-year-career-2026-2028.md)

---

## 2026 下半年（6月-12月）

### 六月 — 入职安顿月

**工作**：
- 06.01 阿里钉钉入职（音频 Agent 方向）
- 第1周：熟悉团队、技术栈、工程规范，了解你的 mentor 风格
- 第2周：开始接手具体任务，和 mentor 对齐期望
- 下班后：LeetCode 每天 2 题，读 1 篇多模态论文
- 了解团队内其他人做的方向，找到可以参与的有意思的项目

**生活**：
- 第1周：在杭州找房子（建议未来科技城附近，离园区近）
- 找到固定的早餐店、健身房/跑步路线、周末咖啡馆（建立第三空间）
- 开始 Duolingo 日语（每天 15 分钟即可）
- 通勤听法语播客维持法语

**感情**：
- 到杭州后自然地告诉婉宁你到了（"我到杭州啦，在找房子"）
- 找好房子后约她出来（具体活动邀约：逛街/吃饭/宝石山徒步）
- 第一次见面轻松为主，不要给压力，带一杯奶茶就够了
- 6月目标：见一面，感受她面对面的状态

### 七月 — 深入实习月

**工作**：
- 争取独立负责一个小模块
- 主动参加团队技术分享，展示你的能力
- 记录所有可量化的成果（准确率、推理速度、用户量等）
- 开始和同期实习生、跨团队同事建立关系

**生活**：
- 周六上午：徒步（宝石山→然后试北高峰、九溪）
- 周六下午：宅着，学日语/画画
- 周日：完全自由
- 每周至少去一次「第三空间」（咖啡馆）

**感情**：
- 如果6月见过一次且氛围好，间隔几天约第二次
- 聊天保持自然频率，她不回消息就等，不要追问
- 注意她的状态：主动找你聊了吗？见面时投入吗？
- 7月目标：见2-3次，从信息交换进入观点共鸣阶段

### 八月 — 成果月

**工作**：
- 做出第一个可量化的成果（这是你简历上最值钱的东西）
- 整理实习笔记，开始把项目经历写成 STAR 格式
- 维护和同事/mentor 的关系（为将来内推做准备）
- 和同期实习生建立群，分享秋招信息

**生活**：
- 继续健身+日语+徒步的节奏
- 如果有兴趣可以尝试一次绘画（Drawabox Lesson 0）
- 8月杭州很热，周末可以试试夜跑或室内活动

**感情**：
- 如果进展顺利：约第三次，可以是稍微深入的活动（看展、做饭、西湖骑行）
- 如果她态度冷淡：退回朋友状态，不要强行推进
- 核心判断：她的行动 > 她的消息回复速度
- 8月目标：明确关系方向（是往恋爱走还是做朋友）

### 九月 — 四个月到期，决定去留

**关键决策**：
- 评估实习成果是否足够（有没有可量化的项目）
- 和 mentor 聊一次：团队对你评价如何？是否建议延长？
- 如果成果不够：考虑延长到 10-11 月
- 如果成果足够：9月结束实习，回校科研

**如果继续实习（10-11月）**：
- 继续做出成果，同时开始准备简历和刷题
- 开始关注 2027 暑期实习的信息

**如果回校**：
- 全力推进科研（选题往多模态/视觉 tokenizer 靠）
- LeetCode 集中刷题（每天 4-5 题）
- 八股文系统整理
- 开始读 2027 暑期实习面经

### 十月-十二月 — 科研+秋招准备期

**工作/科研**：
- 科研推进：争取论文有进展（投一个会议/期刊）
- 每周 3-5 小时读多模态核心论文
- 把实习代码整理成 GitHub repo（开源价值 > 竞赛）
- LeetCode 累计刷到 200-300 题
- 八股文笔记整理完成

**生活**：
- 继续日语（如果还在坚持的话）
- 法语不要丢：每周 1-2 次法语角或法语播客
- 徒步可以试试西安周边（华山、太白山）

**感情**：
- 如果在杭州期间关系有进展：异地后保持联系频率
- 如果没有明确关系：不要因为离开杭州就焦虑，顺其自然
- 校园里也有机会：法语角、课题组交叉合作
- 12月目标：给自己一个诚实的评估，是继续推进还是放下

## 2027 年

### 一月-二月（寒假） — 集中冲刺期

- LeetCode 每天 4-5 题，集中刷
- 八股文系统复习（Transformer、扩散模型、SD、LoRA、优化器）
- 简历最终版完成，找 2-3 个人 review
- 科研继续推进
- 寒假可以安排一次徒步/旅行

### 三月-五月 — 2027 暑期实习申请期 ★★★★★

- **这是你拿 return offer 的关键窗口**
- 集中投递暑期实习：字节/腾讯/阿里/快手/智谱 AI 岗
- 通过阿里同事/牛客网获取内推
- 面试准备：项目讲述练到脱口而出
- 每次面试后复盘，查漏补缺

### 六月-八月 — 暑期实习 ★ 最关键的 3 个月

- **目标：拿到 return offer**
- 选一个和你方向对口的团队（视频生成/多模态）
- 做出有分量的成果
- 和 mentor 建立深度关系
- 7-8月同时参加提前批

### 七月-十二月 — 提前批 + 秋招

- 7-8月：提前批投递
- 9-12月：正式秋招
- 如果有 return offer：有底气谈更好的
- 如果没有：全力冲刺
- 同时科研要继续，论文往"视觉 tokenizer + 下游任务"靠

### 全年生活节奏

- **健身**：保持每周 3 次以上
- **日语**：如果 2026 年坚持了 Duolingo，2027 年可以进阶（Anki + 简单视频）
- **法语**：维持 C1，通勤/散步时听法语内容
- **徒步**：每两周一次
- **社交**：每月至少参加一次非工作社交活动

### 全年感情方向

- 如果和婉宁有进展：保持联系，争取同城机会（如果她也在杭州）
- 如果没有进展：不要执着，校园里主动参加活动认识新的人
- 法语角是你的核武器场景
- 核心：让三个月后的你比现在多认识了三个人、多尝试了一个爱好
