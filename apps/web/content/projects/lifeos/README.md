## 项目背景

lifeOS — 一个个人生活操作系统。

用结构化的方式管理愿景、年/季/月/周/日计划、项目和日历事件。技术上基于 Vite 6 + React 19 + TypeScript 5 + Tailwind v4，内容用 Markdown 驱动，通过 Velite 构建为 JSON。

## 技术栈

- 框架：Vite 6 + React 19 + TypeScript 5
- 样式：Tailwind CSS v4
- 内容：Velite（Markdown → JSON）
- 日历：FullCalendar
- 部署：GitHub Pages (gh-pages)
