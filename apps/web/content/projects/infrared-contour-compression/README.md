## 项目目标

在极端低码率约束下实现轮廓图像/视频的有效压缩与传输。

## 核心需求指标

| 指标 | 目标值 | 说明 |
|------|--------|------|
| **带宽** | 2.4 kbps (2400 bit/s) | 传输信道带宽上限 |
| **分辨率** | CIF 352×288 | 通用中间格式 |
| **帧率** | 10 fps | — |
| **I 帧压缩率** | < 0.01 bpp | 关键帧轮廓图像，每像素不到 0.01 bit |
| **视频平均压缩率** | ≈ 0.002 bpp | 含 P/B 帧的视频平均 |

### 指标推导

- CIF 单帧像素数：352 × 288 = 101,376 px
- 10fps 每秒像素数：1,013,760 px/s
- 2400 bit/s ÷ 1,013,760 px/s ≈ **0.00237 bpp**（视频平均）
- I 帧预算约为平均的 5 倍 → 约 **0.01 bpp**

## 技术路线

1. **先解决关键帧（I 帧）压缩** — 轮廓图像的静态压缩是基础
2. 再扩展到帧间预测（P/B 帧），利用时序冗余进一步降低码率
