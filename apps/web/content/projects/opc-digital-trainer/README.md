## 项目背景

OPC 数字人私教项目，将数字人技术应用于健身私教场景，打造能够提供动作指导、实时纠错、个性化训练计划的数字人私教产品。

### 目标

构建一个数字人私教系统，能够：
- 提供标准动作示范与指导
- 实时识别用户动作并纠错
- 根据用户情况生成个性化训练计划
- 通过数字人形象进行自然交互
