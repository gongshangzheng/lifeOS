## 本周主题

入职 + 找房 + 建立杭州生活与工作节奏。

## 本周核心目标

1. 顺利完成钉钉入职，不出大差错
2. 找到可接受的住处，尽快安顿下来
3. 建立第一周的靠谱形象
4. 自然开启和婉宁的线下推进前奏

## 本周重点任务

### 实习/工作
- [ ] 06.01 完成报到、账号、设备、权限等基础事项
- [ ] 搞清楚团队做什么、你做什么、mentor 是谁
- [ ] 记录团队常用工具、代码仓、沟通节奏
- [ ] 主动和 mentor / 同组同学打招呼，建立第一印象

### 生活/安顿
- [ ] 本周内确定住处
- [ ] 测试通勤路线（时间、地铁/班车、是否堵车）
- [ ] 熟悉园区食堂、便利店、附近生活配套
- [ ] 买齐基本生活用品

### 学习/成长
- [ ] LeetCode 每天 2 题
- [ ] Duolingo 每天 15 分钟
- [ ] 通勤或睡前听法语内容，保持语感

### 社交/感情
- [ ] 自然地告诉婉宁你到杭州了
- [ ] 不急着邀约，先把找房和安顿做好
- [ ] 关注杭州法盟和豆瓣法语学习小组
- [ ] 下载 Tandem，看看杭州本地法语语伴情况

## 风险提醒

- 刚入职时信息很多，不要急着什么都懂，先把最基础的流程走顺
- 找房不要拖太久，住得不稳会直接影响前两周状态
- 这周不适合高强度推进感情，先把自己站稳

## 周末检查点

- 我是否已经稳定住下？
- 我是否知道自己下周要做什么？
- 我是否给团队留下了靠谱的印象？
- 我是否顺利开启了和婉宁的下一步？
