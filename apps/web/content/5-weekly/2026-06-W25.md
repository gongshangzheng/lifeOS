# 周报 - 2026年6月第25周 (06.15 - 06.21)


> 上层： [2026-06 月报](../4-monthly/2026-06.md) ｜ [2026 Q2 季报](../3-quarterly/2026-Q2.md)

## 本周主题

深入项目，开始从“新人”转向“能做事的人”。

## 本周核心目标

1. 开始独立推进一个小任务
2. 认识同期实习生和身边同事
3. 推进第一次线下见面

## 本周重点任务

### 实习/工作
- [ ] 独立完成一个小任务或子模块
- [ ] 主动同步进度，减少 mentor 的管理成本
- [ ] 记录可量化成果的苗头（如效率、效果、实验结果）

### 生活/成长
- [ ] 保持 LeetCode、日语不断线
- [ ] 周末参加一次法语活动或去感受一个新社交场景
- [ ] 试一次轻运动/徒步

### 社交/感情
- [ ] 周末约婉宁出来一次
- [ ] 观察她的状态：愿不愿见面、见面时是否投入、结束后是否自然延续聊天
- [ ] 把目标放在“了解真实相处感”，不是急着定性关系
