**差异化标签**：懂生成模型、能解决真实世界多媒体问题的人。

## 三、技术方向

**第一优先：多模态大模型 + 视频生成**（双主线并行）

- 你的压缩研究和多模态有天然交叉点（视觉 token 化 VQ-VAE/FSQ）
- 扩散模型经验可直接迁移到视频生成
- 2026-2028 有明确商业化窗口

**备选：AI Infra**（需补 C++/CUDA）

**不建议主攻**：自动驾驶、AIGC 应用层、具身智能

**学习路径**：
- 自学视觉编码器（SigLIP/InternViT）和视觉 tokenizer
- 每周 3-5 小时读多模态核心论文
- 不需要上网课考证
- 科研选题往"视觉 tokenizer + 下游任务"靠拢

## 四、就业路径与薪资

**应届硕士算法岗薪资速查**（税前总包，一线）：

| 类型 | 应届 | 3-5年 | 5-8年 |
|------|------|-------|-------|
| 大厂 | 30-50万（SP可达60万+） | 80-130万 | 140-250万 |
| AI独角兽 | 40-70万 | 60-150万 | 100-300万+ |
| 银行/国企 | 25-35万 | 35-45万 | 45-60万 |

**跨公司职级对照**：P5（阿里）/ 1-2（字节）/ 4-5（腾讯）/ 14级（华为）= 硕士应届起点
SP 比白菜高 15-30%，SSP 高 40-60%+签字费

## 五、阿里钉钉实习策略

你的实习方向是**音频 Agent**。

**钉钉在阿里的位置**：2026年3月并入新成立的 ATH 事业群（通义实验室+千问+悟空+MaaS），正从协同办公向 AI 原生工作平台转型。

> 钉钉组织架构、AI团队分布、技术栈详见 → [组织架构](../appendix/org-structure-dingtalk-huawei.md)

### 音频 Agent 和你的目标方向的关系

音频 Agent 涉及语音交互、TTS、语音理解等技术。虽然和你目标的多模态/视频生成不是完全对口，但有几个价值点：
- Agent 架构经验（LLM + 工具调用 + 多模态交互）在算法岗是通用技能
- 阿里平台背书 > 实习方向是否对口
- 2027 年暑期实习可以找更对口的（视频生成/多模态方向）

**不需要因为方向不完全对口而焦虑。** 两段实习经历（零一智能 + 阿里）已经足够有说服力。2027 年暑期实习才是决定性的，那时候可以选择完全对口的团队。

> 音频 Agent 方向详细分析见 → [音频Agent与华为实习分析](../appendix/audio-agent-and-huawei.md)

### 实习目标

1. 做出可量化的成果
2. 争取独立负责一个模块
3. 建立人脉（为 2027 年内推准备）
4. 了解阿里的 Agent 技术栈和工程规范

### 实习目标

1. 做出可量化的成果（这是简历上最值钱的东西）
2. 争取独立负责一个模块
3. 建立人脉（为 2027 年内推准备）
4. 了解阿里的 Agent 技术栈和工程规范

### 转正策略

**日常实习没有转正答辩。** 和暑期实习不同，日常实习不直接关联转正名额。所以：
- 不要花精力准备转正答辩，把精力放在**做出成果**上
- 实习的核心价值 = 可量化的项目经历 + 阿里品牌背书 + 人脉
- 2027 年的暑期实习才是拿 return offer 的机会
- 但仍然建议和 mentor 保持好关系，阿里内部内推非常有价值

- **爱好**：日语（Duolingo 起步）、绘画（Drawabox）、徒步（两步路 APP，宝石山线）
