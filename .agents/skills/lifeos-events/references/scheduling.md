# 日历分类与调度规范

## 日历映射表

| 日历 | 含义 | 任务类型 | 调度规则 |
|------|------|---------|--------|
| **La Réunion** | 会议 | 固定时间会议、面试、周会 | **不可移动**，其他任务让位 |
| **Les cours** | 课程 | 学校课程 | **不可移动** |
| **La Tâche** | 任务 | 有 deadline 的作业、项目 | 时间块，有截止日期 |
| **L'Etude** | 学习 | 读论文、看教程、LeetCode | 时间块或 TODO |
| **La Joie** | 快乐 | 娱乐、兴趣、运动（读书、吉他、旅行、跑步） | 健康运动是硬约束 |
| **La Galère** | 煎熬 | 机械性杂务（填表、跑手续） | TODO 优先 |
| **L'Alert** | 纯粹娱乐 | 桌游、聚会、soirée | 不能太多，适度 |
| **Le Vie** | 生活 | 睡觉、通勤、吃饭、发呆 | 时间块 |
| **La Gestion** | 管理 | 生活管理（找房、搬家） | TODO 或时间块 |

## 调度优先级

1. **La Réunion + Les cours**：固定时间，不可移动，其他一切让位
2. **La Joie（运动/健康）**：硬约束，不可被挤占（跑步至少1小时）
3. **La Tâche**：有 deadline，优先安排
4. **L'Etude**：学习时间，尽量保证
5. **La Galère**：杂务，有空就做
6. **La Joie（非运动）**：娱乐放松，适度
7. **Le Vie / La Gestion**：生活管理，灵活安排

## 健康硬约束

| 习惯 | 频率 | 时长 | 建议时段 |
|------|------|------|---------|
| 跑步 6km | 每天 | **至少 1 小时**（含热身+拉伸） | 早晨或傍晚 |
| 轻食 | 每天 | — | 正常就餐时间 |
| 健身房 | 每 2-3 天 | 1-1.5 小时 | 傍晚或晚上 |

- 健康时间块**不可被其他任务挤占**
- 跑步和健身房不要安排在同一时段
- 健身日可以适当减少当天其他任务量
- **饭后不能马上跑步**，至少等 30 分钟

## 任务分类

### 时间块任务（Time Block）
需要固定时间段、有明确起止时间的任务。写入日报的「时间安排」表 + Google Calendar。

典型：会议、上课、面试、飞机、深度工作、运动、吃饭、通勤。

### TODO 任务
只需要"今天做"，不需要固定时间的任务。写入日报的「今日优先」列表。

典型：发邮件、回消息、买东西、整理文件、背单词。

### 优先级标记
- **P0**：必须今天完成，不能推迟
- **P1**：应该今天完成，可以推迟到明天
- **P2**：最好今天完成，推迟也无妨
- **P3**：有空就做
